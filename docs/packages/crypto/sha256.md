# SHA256

## Overview

SHA256 returns the SHA-256 digest of data as lowercase hex (crypto.createHash('sha256')).

**Node.js equivalent:** `crypto.createHash('sha256').update(d).digest('hex')`

## Signature

```go
func SHA256(data []byte) string
```
